配套工具访问指南

ftp://192.168.1.102/

请按照以下指南使用Internet Explorer或Microsoft Edge访问FTP服务器：

Internet Explorer 操作方法

  1. 打开Internet Explorer浏览器
  2. 在地址栏输入：ftp://192.168.1.102/
  3. 按Enter键访问，完成！

Microsoft Edge 操作方法

  1. 打开Edge浏览器设置，导航到"默认浏览器"设置
  2. 将"允许在Internet Explorer模式下重新加载网站"切换为允许，然后重启浏览器
  3. 重启后，重新打开设置，在"Internet Explorer模式页面"部分添加新网站
  4. 输入ftp://192.168.1.102/并保存
  5. 在地址栏输入ftp://192.168.1.102/访问，完成！
